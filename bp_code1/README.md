## Чиним заказы — тренажёр 1. Соберите запись с get()

Вы уже нашли строку, на которой скрипт падает, и поняли причину: программа слишком жёстко пытается взять order_id, хотя у одного из заказов этого поля нет.
Исправьте строку так, чтобы скрипт больше не падал на таком заказе. На этом шаге главное — сделать получение значения аккуратнее через get()

### Начальный код задания
```python
orders = [
    {"order_id": "A-1001", "item": "Наушники"},
    {"item": "Клавиатура"},
    {"order_id": "A-1003", "item": "Мышь"},
]

for order in orders:
    order_id = order["order_id"]  # исправьте эту строку
    print(f"Заказ {order_id} отправлен в обработку")

print("Обработка завершена")
```

### Эталонное решение эксперта
```python
orders = [
    {"order_id": "A-1001", "item": "Наушники"},
    {"item": "Клавиатура"},
    {"order_id": "A-1003", "item": "Мышь"},
]

for order in orders:
    order_id = order.get("order_id")
    print(f"Заказ {order_id} отправлен в обработку")

print("Обработка завершена")

```
### Ожидаемый ответ
```python
[
    "Заказ A-1001 отправлен в обработку",
    "Заказ None отправлен в обработку",
    "Заказ A-1003 отправлен в обработку",
    "Обработка завершена"
]
```

### Подсказка:
Если ключа в словаре может не быть, используйте `get()`. У этого метода можно сразу указать запасное значение, которое вернётся вместо ошибки.

### Тест:

```
import subprocess
from pathlib import Path

code = Path("main.py").read_text(encoding="utf-8")

assert 'order.get("order_id")' in code or "order.get('order_id')" in code, \
    "Используйте get(), чтобы программа не падала, если в заказе нет order_id."

result = subprocess.run(
    ["python", "main.py"],
    capture_output=True,
    text=True,
    encoding="utf-8"
)

assert result.returncode == 0, "После исправления программа должна выполняться до конца."

output = result.stdout.strip().splitlines()

expected = [
    "A-1001",
    "None",
    "A-1003",
    "Обработка завершена"
]

assert output == expected, "Проверьте, что строка исправлена через get() и программа больше не падает."
```
