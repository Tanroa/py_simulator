## Чиним код
Теперь перед вами полный код того же скрипта обработки заказов. На прошлом шаге вы уже сделали строку безопаснее через `get()`.
Теперь доработайте эту же строку до финального варианта, который Дима проговаривает в квизе: если у заказа нет `order_id`, должно подставляться значение `"без номера"`. После этого запустите скрипт и проверьте, что он доходит до конца.

### Время прохождения - 7 минут

### Начальный код задания
```python
orders = [
    {"order_id": "A-1001", "item": "Наушники"},
    {"item": "Клавиатура"},
    {"order_id": "A-1003", "item": "Мышь"},
]

for order in orders:
    order_id = order.get("order_id")  # доработайте эту строку
    print(f"Заказ {order_id} отправлен в обработку")

print("Обработка завершена")
```

### Эталонное решение эксперта
```python
orders = [
    {"order_id": "A-1001", "item": "Наушники"},
    {"item": "Клавиатура"},
    {"order_id": "A-1003", "item": "Мышь"},
]

for order in orders:
    order_id = order.get("order_id", "без номера")
    print(f"Заказ {order_id} отправлен в обработку")

print("Обработка завершена")

```

### Подсказка:
В прошлом тренажёре вы уже использовали `get()`. Теперь нужно добавить в скобках второе значение через запятую: строку `"без номера"`.

### Тест:

```
import subprocess
from pathlib import Path

code = Path("main.py").read_text(encoding="utf-8")

assert 'order.get("order_id", "без номера")' in code or \
       "order.get('order_id', 'без номера')" in code, \
       'Нужно доработать строку до вида order.get("order_id", "без номера").'

result = subprocess.run(
    ["python", "main.py"],
    capture_output=True,
    text=True,
    encoding="utf-8"
)

assert result.returncode == 0, "После исправления программа должна выполняться до конца."

output = result.stdout.strip().splitlines()

expected = [
    "Заказ A-1001 отправлен в обработку",
    "Заказ без номера отправлен в обработку",
    "Заказ A-1003 отправлен в обработку",
    "Обработка завершена"
]

assert output == expected, "Проверьте, что добавлено запасное значение 'без номера' и вывод совпадает."
```
